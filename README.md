# artist-portfolio
A themed portfolio website showcasing my digital illustration work under the theme "Master Illustrator Registry." Built with HTML, CSS, and JavaScript, it features a responsive multi-page interface, dynamic artwork rendering, automatic featured work updates, searchable gallery, commission management, and an immersive dossier-inspired UI.
